Структура проекта

* web - точка входа и статика
* vendor - внешние модули
* controllers - контроллеры
* views - шаблоны отображения страниц
* core - ядро системы
* runtime - каталог для всего что генерируется по ходу выполнения кода
* exceptions - исключения
* mappers - датамаперы
* models - модели
